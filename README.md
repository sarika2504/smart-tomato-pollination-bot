#  Smart Tomato Pollination Bot

An AI-powered autonomous robot designed for greenhouse tomato farming. The robot detects tomato flowers using computer vision and performs artificial pollination using a vibration mechanism.

##  Project Overview

The Smart Tomato Pollination Bot was developed as a hackathon project to improve pollination efficiency in greenhouse environments where natural pollination is limited.


## Features

-  Automatic tomato flower detection
-  OpenCV image processing
-  Raspberry Pi-based AI
-  Arduino motor control
-  Autonomous robot navigation
-  Vibration-based artificial pollination
-  Greenhouse automation

##  Project Workflow

1. Capture image using Raspberry Pi Camera
2. Detect tomato flower using OpenCV
3. Calculate flower position
4. Navigate robot
5. Activate servo
6. Perform vibration pollination
7. Search for next flower
